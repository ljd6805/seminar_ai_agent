## 한눈에

- **목표**: LLM·Agent·Skill/MCP/Harness를 구분해 이해하고, 세미나 후 **검증 가능한 작은 업무 하나**에 적용해 보게 하기
- **청자**: 반도체 설계/검증/소프트웨어 엔지니어, AI Agent 이해도 낮음
- **발표 시간**: 90분
- **핵심 메시지(고정)**: *AI Agent는 사람의 판단을 대체하는 도구가 아니라, 판단 전에 필요한 반복 작업을 줄이는 도구다*
- **관통 시나리오(v1.8)**: **Regression Triage Agent** — 하나의 작업을 데모 1(prompt만) → 데모 2(Skill) → 데모 3(MCP) → 데모 4(Harness) → 사람 승인으로 단계적으로 키움
- **4요소**: Agent(실행 시스템) · Skill(작업 매뉴얼) · MCP(공용 규격) · Harness(통제 구조) · **4원칙**: 작게 시작 · 출력 구조화 · 근거 남기기 · 사람 승인
- **자료 형태**: reveal.js 단일 파일 HTML — 키보드 네비게이션, 진입 시 자동 스태거(fragment 없음), ESC 오버뷰, DEMO/READ/APPENDIX 레이어 배지
