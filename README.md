## 한눈에

- **목표**: LLM·agent·skill/MCP/harness를 구분해 이해하고, 세미나 후 **검증 가능한 작은 업무 하나**에 적용해 보게 하기
- **청자**: 반도체 설계/검증/소프트웨어 엔지니어, AI agent 이해도 낮음
- **발표 시간**: 90분
- **핵심 메시지(고정)**: *AI agent는 사람의 판단을 대체하는 도구가 아니라, 판단 전에 필요한 반복 작업을 줄이는 도구다*
  — agent = LLM+도구+루프 실행자 · skill = 일하는 방법 · MCP = 시스템 연결 · harness = 깨지지 않게 감싸기
- **자료 형태**: reveal.js 단일 파일 HTML — 키보드 네비게이션, 진입 시 자동 스태거(fragment 없음), ESC 오버뷰 그리드
