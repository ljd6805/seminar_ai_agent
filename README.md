## 한눈에

- **목표**: AI agent를 처음 소개하고 "나도 써봐야겠다"는 동기 부여
- **청자**: 반도체 설계/검증 담당자, AI agent 이해도 낮음
- **핵심 서사**: AI agent가 있다 → 이런 장점이 있다 → 이렇게 쓰면 된다
- **자료 형태**: 가로 슬라이드 HTML, 키보드 네비게이션, 애니메이션

### 전체 흐름

```
도입    AI agent란 게 있다 (+ 오프닝 데모 Ex1: 이 자료를 만든 대화 과정)
본문 1  사내/사외 정보 흐름   — 우리 환경부터
본문 2  plugin / skill / mcp — 부품을 이해한다 (+ Ex2: 문서 parsing 기본 동작)
본문 3  harness              — 틀로 감싸 제대로 쓴다 (+ Ex4: 같은 작업의 harness 유무 대조)
마무리  (보류)
```

## 진행 상태

**확정** — 핵심 서사, 본문 3파트 구조, 오프닝 데모 배치, 데모는 미리 만든 예시(문서 parsing 공유), Ex4 대조 축은 품질·재현성

**미결** — 대표 plugin 리스트 노출 범위(페이지 수), 마무리 파트

**보류** — 모든 시각 표현·디자인·애니메이션 구체화
